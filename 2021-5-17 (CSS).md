### CSS简介

```html
	CSS:层叠样式表(英文全称：Cascading Style Sheets)是一种用来表现HTML（标准通用标记语言的一个应用）或XML（标准通用标记语言的一个子集）等文件样式的计算机语言。CSS不仅可以静态地修饰网页，还可以配合各种脚本语言动态地对网页各元素进行格式化。 
	CSS 能够对网页中元素位置的排版进行像素级精确控制，支持几乎所有的字体字号样式，拥有对网页对象和模型样式编辑的能力。

CSS两大功能：
1. 控制元素外观
2. 控制元素位置（布局）

两种布局方式
float布局
position布局
CSS3新增的:flex布局

文档流：
是html元素按顺序进行排列显示，这个顺序是从左到右，从上到下。也叫标准流。
布局前提：
浏览器交出页面布局的权限，交给用户。即CSS所控制元素将从文档流脱离出来。而脱离出来的元素一定会变成块级元素。
脱离文档流的手段：浮动，绝对定位
```

### 浮动布局

```html
浮动布局：
<head>
    <style>
        .wrapper{
            /* 大模块没有被更大的模块包裹，则百分比以整个界面作为参照 */
            width: 100%;
            margin: 0 auto; 
            text-align: center;
        }
        .header{
            width: 100%;
            height: 90px;
            background-color: cyan;
        }
        .main{
            /* 定义大的模块标准 */
            width: 100%;
            height: 1000px;
        }
        .left{
            /* 小模块，百分比按照大模块main设置的初始值作为参照 */
            width: 10%;
            height: 100%;
            background-color: darkgoldenrod;
            float: left;
        }
        .content{
            width: 80%;
            height: 100%;
            background-color: darkgray;
            float: left;
        }
        .aside{
            width: 10%;
            height: 100%;
            background-color: darkgreen;
            float: left;
        }
        .footer{
            /* 没有float: left只有height无法显示背景颜色？(前面float之后后边的也需float?) */
            width: 100%;
            height: 50px;
            background-color: rgb(48, 51, 221);
            float: left;
        }
    </style>
</head>
<body>
    <div class="wrapper">
        <div class="header">
            header
        </div>
        <div class="main">    
            <div class="left">
                左侧
            </div>
            <div class="content">
                内容区
            </div>
            <div class="aside">
                侧边栏
            </div>
            <div class="footer">
                脚
            </div>
        </div>
    </div>
</body>
```

### position布局

```html
position布局：
position:relative:相对地址(配合float调相同位置块与块会覆盖)
position:absolute:绝对地址(配合float调相同位置块与块不会覆盖，自动补全空白位置)
```
